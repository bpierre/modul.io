Install modul.io
================

## Instructions

Create a directory:

    mkdir modul.io && cd modul.io

Clone the repository:

    git clone git://github.com/bpierre/modul.io.git

Install dependencies:

    npm install .

Launch modul.io:

Node:

    node src/server.js

Or NPM:

    npm start

If installed with `npm install -g`:

    modul.io
